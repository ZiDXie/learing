## 打开root用户自动source

```bash
sudo vim /root/.bashrc
```

在文件末尾添加以下内容：

```bash
source /home/<your_username>/.bashrc
```

## rosip自动切换

```bash
export ROS_IP=$(ip -4 addr show scope global | grep -oP '(?<=inet\s)\d+(\.\d+){3}' | head -n 1)
if [ -z "$ROS_IP" ]; then
    export ROS_IP="127.0.0.1"
fi
```
